Amnesia Pass Generator

MD5 existe apenas para compatibilidade e não é recomendado para novas senhas.

Dica: instale o app para uso offline (menu do navegador → Instalar app).

Sobre a aplicação

O Amnesia Pass Generator é um gerador de senhas determinístico. Ele cria hashes iterativos baseados em uma palavra-chave que você fornece.

Como funciona?

1. Você insere uma palavra-chave.
2. O sistema gera um hash (SHA256 ou SHA512; MD5 está disponível mas não é recomendado).
3. O processo se repete pelo número de iterações definido.
4. O resultado final é cortado para o tamanho desejado.
5. Prefixo e sufixo (se informados) são adicionados ao resultado final.

Segurança e Privacidade

Todo o processamento é feito exclusivamente no seu navegador (Client-Side). Nenhuma informação, palavra-chave ou senha gerada é enviada para servidores externos.

⚠️ Aviso sobre Iterações:

Como o processamento é local (usa o processador do seu dispositivo), definir um número excessivamente alto de iterações (ex: milhões) pode causar lentidão ou travamento temporário do navegador. Para uso comum, valores baixos (1 a 1000) são suficientes e instantâneos.

Dica de Uso

Use os campos de Prefixo e Sufixo para adicionar caracteres especiais e letras maiúsculas sem alterar o hash base.
Exemplo: #MeuPrefixo + [hash gerado] + #MeuSufixo

Boas práticas

1) Use uma seed/senha mestra forte (frase longa e pouco óbvia) e um salt diferente para cada serviço.
2) Se usar a versão CLI, cuide com histórico de terminal e clipboard: evite deixar a seed/salt expostas e limpe conteúdos sensíveis.
